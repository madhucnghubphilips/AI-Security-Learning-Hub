# LLM09 - Misinformation Healthcare AI Security Lab

Full Streamlit UI demo for OWASP LLM09 - Misinformation with healthcare examples and Gandalf-style challenge levels.

## Includes
1. What is LLM09 - Misinformation
2. Vulnerable AI Behavior
3. Secure AI Behavior
4. Side-by-Side Comparison
5. Interactive Challenge Levels
6. Mitigation Strategies and Security Best Practices - including LLM01 Prompt Injection controls

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

Windows: run `run_windows.bat`
Linux/macOS: run `chmod +x run_linux_mac.sh && ./run_linux_mac.sh`
