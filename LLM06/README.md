# LLM06 - Excessive Agency Healthcare Demo

A full Streamlit UI demo for OWASP Top 10 for LLM Applications: **LLM06 - Excessive Agency**.

## Features

- Vulnerable Healthcare AI simulation
- Secure Healthcare AI simulation
- Side-by-side comparison
- Gandalf-style interactive levels with Next button
- Answers loaded from `Answers.txt`
- Validation of user responses against answer keywords
- Healthcare examples covering synthetic PI/PHI scenarios across ultrasound, patient monitoring, image-guided therapy, diagnostic informatics, MRI, and CT

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Note

All names, patient IDs, accession numbers, device IDs, and PHI examples are synthetic for education and awareness only.
