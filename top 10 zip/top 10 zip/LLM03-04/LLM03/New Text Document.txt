Recommended demo flow

Start with Guardrails Disabled, run the attack, show the manipulated output and event timeline.

Then enable Guardrails / Mitigations and run again to show how controls such as allowlisting, checksum verification, source validation, egress control, and output scanning reduce the risk.

Safety note: this is a local-only simulation. It does not steal secrets, call external servers, exploit systems, or perform real malicious behavior.
