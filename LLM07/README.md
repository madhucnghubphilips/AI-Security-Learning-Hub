# LLM07 - System Prompt Leakage Healthcare Demo

A downloadable Streamlit end-to-end attack simulation for OWASP Top 10 for LLM Applications: **LLM07 - System Prompt Leakage**.

## Features

- Vulnerable Healthcare AI demo
- Secure Healthcare AI demo
- Side-by-side comparison
- Gandalf-style interactive levels: Easy, Medium, Difficult
- Healthcare real-time style examples: ultrasound, patient monitoring, CT, clinical workflow notes, support debugging, audit pretexts
- Interactive validation using facilitator answers in `Answers.txt`
- No external API or model required

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Training Usage

Use the interactive levels for awareness sessions. Participants should identify the weakness and explain how the secure assistant should respond.
